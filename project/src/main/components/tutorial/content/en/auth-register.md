# Register

Create a new account so you can save your own data, locations, and region presets later.

## Suggested Steps

- Fill in your username, email address, and password first.
- Confirm the password again before submitting.
- After a successful sign-up, the page will return to the login view automatically.

## Tips

- Username, email, and password are validated on the client side first.
- If the system says the account already exists, switch to another username or email address.
