# 粵拼轉 IPA

這個工具用來批量把 Excel 內的粵拼轉成 IPA。實際流程是：登入後上傳檔案、伺服器排程處理、完成後下載新檔。

## 先知道的限制

- 只接受 `.xlsx`、`.xls`。
- 單檔上限 3 MB。
- 沒登入時，拖放或選檔都不會開始轉換。

## 建議操作順序

1. 先按設定卡片打開規則視窗。
2. 視需要調整替換規則，並按儲存；也可以匯出 JSON 備份，或匯入既有設定。
3. 回到主畫面上傳 Excel。
4. 等待進度條跑完。
5. 先看預覽表，再下載結果檔。

## 實際輸出

- 完成後會顯示前 10 筆預覽。
- 預覽欄位包含：字、粵拼、IPA、聲母、韻母、音調。
- 下載檔名會保留原始檔名，但前面加上工具的結果前綴。

## 設定規則的重點

- 規則視窗有分頁，可依分類檢查替換項。
- 匯入 JSON 只是把規則載入目前畫面；若想下次也沿用，記得再按一次儲存。
- 重設會把規則恢復成預設值，並清掉本地保存的自訂規則。

## 常見坑位

- 匯入成功但沒按儲存，重新整理後會回到舊設定。
- 檔案格式不對或太大，會直接報錯，不會進入處理佇列。
- 處理失敗時工具會重置任務狀態，需重新上傳。
