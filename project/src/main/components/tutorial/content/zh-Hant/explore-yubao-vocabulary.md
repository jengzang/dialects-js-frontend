# 語保詞彙

語保頁面實際是同一個介面分成「詞彙」與「語法」兩個分頁。詞彙頁對應網址查詢參數 `?tab=vocabulary`；如果網址沒有 `tab`，系統進頁時會自動補成詞彙分頁。

## 建議操作順序

1. 先確認自己在「詞彙」分頁。
2. 在輸入框打詞，但不要急著按 Enter；先從下拉建議中點選完全符合的項目。
3. 依需求切換顯示方式：表格、卡片、地圖。
4. 想快速瀏覽全部可按右上角「查看全部」，再從彈窗挑一筆。

## 三種顯示方式的差別

- 表格：只有當輸入內容和詞庫中的某個詞完全一致時，才會真正顯示表格。
- 卡片：有效輸入後會向後端取資料，支援前端再加一層關鍵字過濾，並用無限捲動逐步載入更多卡片。
- 地圖：同樣依賴有效輸入；若資料尚未載入，切到地圖時會自動補抓資料。

## 實際輸出

- 表格模式會把 `word` 當成預設篩選條件。
- 卡片會顯示地點鏈、分區鏈、讀音與字詞說明。
- 「查看全部」彈窗列出所有已載入的詞條，方便初學者先挑再看。

## 常見坑位

- 只輸入半個詞或自由輸入未在清單中的內容，表格不會出現。
- 表格模式提示「請選建議」時，通常不是當機，而是你還沒點選完整建議詞。
- 切換分頁只會改網址中的 `tab`；表格 / 卡片 / 地圖模式本身不會寫進網址。
