# 廣東自然村表格

這一頁用通用表格元件顯示「广东省自然村」資料，適合做欄位篩選、排序、分頁閱讀與關鍵字查找。表格資料來源不是一次全部灌進前端，而是依目前條件向後端查詢。

## 這一頁有什麼

頁面結構很簡單：
- 上方只有標題。
- 下方是一個可互動的大表格。
- 表格上方有搜尋框。
- 表頭可排序；部分欄位可打開篩選器。
- 下方有分頁控制與全螢幕按鈕。

## 欄位說明

這一頁固定顯示 8 欄：
- 市級
- 區縣級
- 鄉鎮級
- 行政村
- 自然村
- 方言分布
- longitude
- latitude

其中可打開篩選器的欄位是：
- 市級
- 區縣級
- 鄉鎮級
- 行政村
- 方言分布

自然村、longitude、latitude 這三欄不能用欄位篩選彈窗，但仍然會參與全表搜尋與排序。

## 建議操作順序

1. 先用上方搜尋框縮小範圍。
2. 再按欄位名稱旁的篩選入口，限定市級、區縣級或方言分布。
3. 需要看集中在哪裡時，按欄位上的 ▲ / ▼ 排序。
4. 資料很多時，用下方分頁切換或直接輸入頁碼。
5. 想一次看更多欄位寬度時，切換全螢幕。

## 搜尋怎麼運作

- 搜尋框會在你輸入後延遲約 300 毫秒再重新查詢。
- 搜尋會把目前設定的關鍵字送到後端，並套用到這張表的全部欄位。
- 只要文字符合任一欄位，就可能出現在結果裡。
- 每次重新搜尋時，頁碼會回到第 1 頁。

如果你輸入後覺得結果沒有立即變化，通常只是因為有短暫防抖，不是失效。

## 欄位篩選怎麼用

可篩選欄位的表頭旁邊會顯示提示圖示。點擊後會打開該欄位的選項清單。

這個篩選器有幾個重要特性：
- 選項不是寫死的，而是依目前其他篩選條件與搜尋文字向後端請求。
- 打開某欄篩選器時，系統會先保留其他欄條件，再取得這一欄可選值。
- 你可以多選。
- 有「全選 / 反選」邏輯。
- 按確認後才真正套用到表格。
- 套用篩選後，頁碼也會回到第 1 頁。

也就是說，如果你先搜尋、再開篩選器，彈窗裡看到的選項已經是目前搜尋條件下仍可能出現的值，不一定是整張表的完整全集。

## 排序怎麼用

每一欄表頭右側都有：
- ▲ 升序
- ▼ 降序

點下去會立即重新查詢資料。排序與搜尋、篩選可以同時生效。

## 分頁怎麼用

- 每頁固定顯示 50 筆。
- 左右箭頭可以上一頁、下一頁。
- 中間會顯示「目前頁 / 總頁數」。
- 點頁碼文字後，可以直接輸入想跳轉的頁數。
- 若輸入超出範圍的頁碼，系統會提示頁碼無效，並維持原頁。

## 全螢幕怎麼用

右下角有全螢幕切換按鈕：
- 進入全螢幕後，表格會以覆蓋層方式顯示，方便看寬欄位。
- 退出後會回到原本頁面位置。
- 全螢幕不會改變搜尋、排序、篩選或頁碼狀態。

## 你會看到哪些輸出

- 載入中時，表格區會顯示 loading 狀態。
- 查不到資料時，表格區會顯示空資料提示。
- 正常查到資料時，會看到目前頁的結果列。
- 所有搜尋、篩選、排序、翻頁操作都會直接影響目前表格內容與總頁數。

## 管理員才會看到的功能

如果目前登入角色是 admin，工具列還會多出：
- 匯出 Excel
- 新增資料
- 編輯模式
- 批量替換
- 提交批量修改
- 每列的刪除按鈕

一般使用者不會看到這些按鈕，所以若你在教學畫面上找不到，不代表頁面壞掉，只是權限不同。

## 常見容易誤會的地方

- 篩選器選項會受目前搜尋與其他欄條件影響，所以不是永遠固定不變。
- 搜尋會搜尋全部欄位，不只是你眼睛正在看的那一欄。
- 「自然村」欄雖然不能開欄位篩選器，但仍可用上方搜尋框搜尋。
- longitude、latitude 主要是資料欄，不是這頁的地圖互動入口。
- 覺得一頁資料太少不是異常；元件固定每頁 50 筆。

## 適合什麼情況使用

- 想快速比對同一欄位在不同地區的資料。
- 想用關鍵字先抓出候選村落，再按行政層級繼續縮小。
- 想做大量掃描、整理或人工校對。
