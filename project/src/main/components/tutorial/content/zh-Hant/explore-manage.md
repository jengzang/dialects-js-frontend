# 表格管理

這一頁是管理用工具，不是一般瀏覽頁。進頁後會先檢查登入與角色；只有管理員能繼續使用。

## 進入後會發生什麼

- 頁面會先做登入驗證，再讀取使用者角色。
- 非管理員會看到無權限提示，並在幾秒後跳走。
- 管理員若之前用過此頁，系統會自動嘗試載入上次使用的資料庫、資料表與本地配置。

## 建議操作順序

1. 先選資料庫，再選資料表。
2. 若清單裡沒有你要的名稱，可切成自訂資料庫或自訂資料表，輸入後失焦套用。
3. 取回欄位後，先用「套用預設」或「全選 / 全不選」做初步配置。
4. 再微調欄寬、是否可篩選，以及預設篩選值。
5. 確認至少勾到一欄後，按「顯示表格」。
6. 若這組設定之後還會再用，記得另外按「儲存配置」。

## 實際輸出

- 顯示表格時，頁面會重新掛載 UniversalTable，所以會以你剛設定的欄位重新載入。
- 顯示表格後，上方只保留收合工具列；需要再改配置時可重新展開。
- 配置儲存、讀取、清除都是以「資料庫 + 資料表」為單位，存在本機 localStorage。

## 常見坑位

- 沒先選資料庫與資料表，就不能存、讀、清配置。
- 一欄都沒選時，按「顯示表格」只會跳警告。
- 沒有內建預設的資料表，系統會改成預設勾前 10 欄。
- 本機清除瀏覽器資料後，保存的配置也會一起消失。
