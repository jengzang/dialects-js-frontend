# 查字

從單個或少量漢字直接反查各地讀音。這一頁最適合新手起步：你只要先知道「要查哪個字」，就能很快看到它在不同地點的資料。

## 你在這一頁會看到什麼

- 上方只有一個漢字輸入框。
- 下方共用一組「地點 / 分區」篩選區。
- 最下面是執行按鈕；按下後不會直接在本頁展開結果，而是跳到「結果頁」。
- 右下角如果看到骰子按鈕，代表可以套用系統給的示例配置。

## 最適合新手的操作順序

1. 先在輸入框中輸入要查的字。
2. 第一次使用時，建議先只輸入 1 個字，不要一開始就查太多字。
3. 如果你只想看少數地點，再到下面補選地點或分區；如果暫時沒有想法，可以先不設分區直接跑一次。
4. 按「運行」後，系統會帶著你的條件跳到結果頁。

## 這一頁有哪些實際規則

- 輸入框不能是空的，否則運行按鈕會保持不可用。
- 漢字有效字數最多只取前 10 個；如果超過，頁面會提示你，超出的有效字不會再繼續加入。
- 地點與分區是所有查詢分頁共用的第二層限制條件：它們不是必填，但會直接影響結果範圍。
- 若你沒有明確選分區，系統仍會用你輸入的地點名稱去查。

## 地點 / 分區怎麼用

- 左邊文字框可直接輸入地點。
- 「選擇地點」按鈕可從彈窗中挑選地點，不必手打。
- 右邊分區區塊可在「地圖分區」和「音典分區」之間切換。
- 底部會顯示目前實際命中的地點數量；如果數量很多，可以先縮小範圍再查。

## 結果頁會看到什麼

- 你在這裡按下運行後，會進入結果頁的「查字結果」模式。
- 若你查的是多個字，建議先挑最關心的地點閱讀，不要一口氣把所有資料都混在一起看。
- 如果結果方向對了，再回來加地點或分區細化；如果方向不對，先回來改字，不要急著加更多限制。

## 新手最常見的卡點

- 按鈕是灰的：通常是因為漢字還沒輸入，或地點元件仍在處理輸入狀態。
- 查到的地點太多：先限制分區，或改成只看幾個具體地點。
- 查到的內容不像你預期：先確認是不是輸入了多個字，或不小心帶著上一輪的地點 / 分區條件。

## 什麼時候改用別的頁面

- 你想從中古條件出發，不是從漢字出發：改用「查中古」。
- 你想從音值倒推來源：改用「查音位」。
- 你想直接比較兩個字在各地的差別：改用「比較漢字」。
