# 字表·切韻

切韻分頁和近古一樣，也是多子表模式；差別在於它用的是切韻相關資料表集合。這裡仍會把目前子表與層級寫進網址，方便直接回到同一視圖。

## 建議操作順序

1. 先選要看的切韻資料表。
2. 套用一組最接近需求的 preset。
3. 只在真的需要時再手動增減層級，避免樹太深不好讀。
4. 用右側搜尋快速定位節點，再決定是否打開註釋輔助閱讀。

## 使用提醒

- 這一頁不是全文檢索頁，主軸仍是按音韻欄位組成樹狀結構。
- 搜尋、註釋開關、重試、快取都沿用共用字表樹邏輯。
- 若網址中的 `table` 或 `levels` 不合法，頁面會回退到此分頁的預設配置。

## 常見坑位

- 子表一換，上一張表的層級鍵可能就失效，所以頁面會自動修正，不一定照網址原值展示。
- 如果你只是想看不同排序，不一定要重配全部層級；先調整現有層級順序通常就夠。
