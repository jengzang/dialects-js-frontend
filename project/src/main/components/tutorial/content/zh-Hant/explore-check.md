# 數據檢查

這一頁適合先把表格檔做一次機器檢查，再決定要不要進一步人工修正。實際頁面會先要求登入，之後才能上傳檔案。

## 你會看到什麼

- 首頁先選資料格式：音典、跳跳老鼠、縣志。
- 也可以選是否把內容轉成簡體再處理；頁面本身偏向建議保留繁體來源。
- 可上傳 `.xlsx`、`.xls`、`.doc`、`.docx`、`.tsv`，單檔上限 3 MB。
- 上傳成功後，頁面會自動分析，不用另外再按一次「開始」。

## 建議操作順序

1. 先確認自己選的格式和原始檔一致。
2. 上傳檔案，等頁面自動完成分析。
3. 先看左側錯誤統計，再點進各類錯誤。
4. 需要時用搜尋、聲母篩選、韻母篩選縮小範圍。
5. 若只是瀏覽問題，可保持一般模式；若要改資料，再切到編輯模式。
6. 編輯完成後批量儲存，系統會重新分析一次最新內容。

## 實際輸出

- 左側會整理錯誤類型、調值統計、聲母與韻母統計。
- 點錯誤列會自動跳到對應資料行。
- 點聲母或韻母統計，會直接套用篩選。
- 編輯模式下可修改欄位，也可標記整列刪除。
- 批量儲存後，前端會逐列送出更新，刪除列也會一起提交，最後重新載入分析結果。

## 常見坑位

- 沒登入時，上傳不會繼續處理。
- 格式選錯，往往不是「少量錯誤」，而是整份資料會被錯誤拆欄。
- 超過 3 MB 或副檔名不符會直接被拒絕。
- 退出編輯模式時，未儲存修改會被清空。
- 批量儲存沒有待改內容時，系統只會提示沒有可存內容。
