# 陽春口語

這一頁本身很直接：主畫面就是一個 UniversalTable，資料來源固定是 `spoken` 資料庫下的「口语字」表。

## 你可以怎麼用

1. 直接在表格裡看欄位資料。
2. 需要縮小範圍時，用可篩選欄位做過濾。
3. 若想連到村落資料，可按標題旁邊的村落按鈕，跳去 `/explore/villages/yc`。

## 主要欄位

- 本字考
- IPA
- 粵拼
- 來源
- 聲母、韻母、音調
- 詞性
- 釋義
- 例詞例句
- 待校及說明

## 使用提醒

- 並不是每一欄都能篩選；像 IPA、粵拼、釋義、例句等在這頁設定成不可篩選。
- 若你是想做大範圍資料配置，不要在這一頁改，應改去表格管理頁處理。
