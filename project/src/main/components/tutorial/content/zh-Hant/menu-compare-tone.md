# 比較調類

用來直接比較兩個調類在各地是否合流。這一頁的規則很單純，但一定要記住：一次只能比較兩個調類。

## 你在這一頁會看到什麼

- 上方是一整塊調類勾選區。
- 每勾到第 1 個和第 2 個調類時，畫面會用不同顏色標示它們。
- 一旦已經選滿 2 個，其它未選調類會暫時變成不可再勾。
- 下方仍然有地點 / 分區設定。
- 按下比較後，會直接跳到地圖頁。

## 最適合新手的操作順序

1. 先勾選第一個調類。
2. 再勾選第二個調類。
3. 確定兩個類別沒有選反後，再補地點 / 分區。
4. 按下比較，去地圖頁看哪些地方已合流、可能合流或未合流。

## 這一頁有哪些實際規則

- 必須剛好選 2 個調類，比較按鈕才會啟用。
- 已選滿 2 個後，其他未選項會自動禁用；你若想換組合，先取消其中一個再重選。
- 執行時會直接呼叫 compareTones，然後把結果轉成地圖顏色圖例。

## 地圖頁上會看到什麼

- 圖例重點通常是：已合流、可能合流、未合流、未知。
- 所以這頁的核心不是看單一調值，而是看兩個調類在各地的關係。

## 新手最常見的卡點

- 按鈕是灰的：通常只選了 1 個，或不小心取消掉其中 1 個。
- 想選第 3 個卻點不動：這是正常設計，因為一次只允許兩個。
- 地圖看起來差很多：先縮小到少數地點，再理解「合流 / 未合流」的意義。

## 什麼時候改用別的頁面

- 想直接查看各地完整調類資料：改用「查調類」。
- 想比較兩個字或兩組中古條件：改用其他比較頁。
