這一頁對應 /menu/pho/custom，用來按自選分類欄位重排音系矩陣。

建議新手這樣用：
1. 先選「聲母 / 韻母 / 聲調」。
2. 輸入地點。
3. 再選橫軸、縱軸、格內項目三個分類欄位。
4. 按查詢後查看下方矩陣與格內細節。

實際規則：
- 這一頁最多可查 10 個地點。
- 可選欄位不是固定的，會跟著你目前偏好的字表而改變。
- 如果目前字表不在支援名單內，頁面不會提供查詢，而是直接顯示不支援提示。
- 頁面會從網址讀取 feature、locations 與三個分類欄位；參數完整且有效時，地點匹配完成後會自動查詢。
- 第一次成功查詢後，頁面會把目前選項同步回網址，方便分享同一組矩陣。
- 使用瀏覽器前進/後退時，只要網址參數和目前結果不同，頁面會更新輸入與欄位選擇，並清掉舊矩陣。
- 查詢結果啟用了格內細節資料，所以不只看得到矩陣，還能看每格對應的細項。

容易踩坑：
- 改了字表之後，可選分類欄位也會跟著換；先前能用的欄位不一定還存在。
- 網址參數若不合法，頁面只會保守回退到可用預設值，不會照單全收。
