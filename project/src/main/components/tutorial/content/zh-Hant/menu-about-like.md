這一頁對應 /menu/about/like，用來關注作者、給專案按星，或打開支持彈窗。

建議新手這樣用：
1. 想追蹤作者近況，就按上方關注按鈕。
2. 想支持開發，就先看看三個 GitHub 專案卡片。
3. 想贊助時，再按下方支持按鈕打開彈窗。

實際規則：
- 上方關注按鈕會直接在新分頁打開作者知乎首頁。
- 中間三張卡片分別連到 frontend、backend、build 三個 GitHub 倉庫，點擊後都會開新分頁。
- 下方支持按鈕不會跳頁，而是把 SupportPopup 彈窗打開。
- 關閉彈窗後，仍會留在同一頁。

容易踩坑：
- 「支持」不是外部連結，而是站內彈窗。
- 這頁偏向支持與關注入口，不提供帳號功能或資料查詢。
