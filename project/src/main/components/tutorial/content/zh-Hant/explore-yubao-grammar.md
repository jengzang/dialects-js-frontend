# 語保語法

語法頁和詞彙頁共用同一個外框，但語法分頁對應的是 `?tab=grammar`。切換分頁時，頁面會同步改網址查詢參數。

## 建議操作順序

1. 先切到「語法」分頁。
2. 在輸入框輸入句式或例句片段。
3. 從下拉建議中點選完整條目，不要只停留在手打內容。
4. 再選擇表格、卡片或地圖模式查看。
5. 若不確定要查哪句，可按「查看全部」從完整清單挑選。

## 顯示邏輯

- 表格模式只有在輸入值與現有句式完全匹配時才會開表。
- 開表後，頁面會把 `sentence` 作為預設篩選條件。
- 卡片與地圖模式也需要有效輸入；有效後前端會自動觸發資料載入。
- 卡片模式支援本地關鍵字再過濾，適合先縮小結果再細看。

## 實際輸出

- 卡片通常會顯示 form_a 到 form_e 串接出的地點層級、語音標註與分類資訊。
- 若資料很多，卡片區會靠無限捲動分批顯示，不會一次全部灌出。

## 常見坑位

- 沒點建議、只留模糊輸入時，表格區會停在提示狀態。
- 切到地圖看不到資料時，先確認輸入是否為有效條目，而不是空字串。
- 詞彙與語法各有自己的輸入框；切分頁後不要以為上一頁輸入會直接變成可用查詢。
