這一頁對應 /menu/about/settings，是「關於」頁裡的設定分頁。

建議新手這樣用：
1. 先選介面語言。
2. 再選你想當作預設的字表。
3. 之後回到音系相關頁面查看變化。

實際規則：
- 這個分頁同時提供兩項設定：語言、預設字表。
- 語言可選 zh-Hant、zh-CN、en。
- 切換語言後，系統會先顯示成功訊息，再在約 0.5 秒後自動重新整理整個頁面。
- 預設字表的選單內容來自 TABLE_COLUMN_SCHEMAS，因此會列出目前程式已註冊的字表。
- 預設字表會寫入全域偏好設定，之後像「音系自定義」這類頁面會依此決定可用欄位與預設分類。

容易踩坑：
- 改語言會自動刷新頁面，不是瞬間無刷新切換。
- 改完預設字表後，某些音系頁的欄位選項會跟著變，這是正常行為。
