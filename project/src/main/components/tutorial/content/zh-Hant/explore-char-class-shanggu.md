# 字表·上古

上古分頁沿用同一套字表樹頁，但它的表別、可用層級與預設組合和中古不同。這裡對應 `/explore/char-class?tab=shanggu`。

## 建議操作順序

1. 先用預設分組進頁，快速看目前樹的主幹。
2. 如果要看韻部系統，就從韻部、韻母、聲調這條組合開始。
3. 如果要看聲母系統，就改用聲母組、聲母、聲調這條組合。
4. 需要更細時，再手動新增或改寫層級。

## 這一頁的實際配置

- 上古頁目前使用 `old_chinese` 表。
- 可用層級包含聲調、聲母、韻母、韻部、聲母組、r介音、非三等、諧聲域。
- 內建 preset 至少有韻系導向與聲母組導向兩組，切換後會整組替換層級順序。
- 註釋顯示、搜尋、重試與網址同步機制，都和其他字表分頁共用同一套行為。

## 使用提醒

- 樹狀資料會按目前 `table + levels` 組合做快取；同組配置切回時通常不必重抓。
- 但切換到別的主分頁後，這頁的本地狀態會先重置，再按路由重新套用。
- 搜尋只縮小目前樹的可見內容，不會替你改變左側層級設定。

## 常見坑位

- 若你手動改成一組不理想的層級順序，最省事的做法通常是直接切回某個 preset。
- 路由參數若缺失或不合法，頁面會自動回退到預設表別與預設層級。
