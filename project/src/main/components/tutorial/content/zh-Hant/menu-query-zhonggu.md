# 查中古

用中古音條件做檢索，按地點整理聲母、韻母、聲調等查詢結果。

![查中古頁面示意](/tutorial/menu-query-zhonggu/overview.svg)

## 建議操作順序

- 先挑卡片、鍵名和鍵值條件。
- 再補充分區或具體地點。
- 運行後到結果頁或地圖頁看輸出。

## 使用提醒

- 這一頁適合從中古地位反查現代方言表現。
- 如果條件很多，先從少量條件開始更容易判斷結果是否合理。

## 補充入口

需要完整示例時，可查看[示例說明頁](https://example.com/tutorial/query/zhonggu)，或直接回到 `/menu/query/zhonggu` 重新整理條件。
