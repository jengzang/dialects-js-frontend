# Praat·音高定調

這一頁是分析結果上的二次操作工具，核心不是自動判斷調類，而是讓你在音高曲線上手動框選樣本，再用石峰 T 值法整理比較。

## 建議操作順序

1. 先確保這次分析結果真的包含 pitch 時序資料。
2. 在第一步的音高圖上框選一段樣本，輸入調名後加入清單。
3. 同一個調名可以重複加入多段樣本；它們會累積在同一調類下。
4. 準備好至少一批調類後，再按分析按鈕生成 T 值圖。
5. 需要留存時可直接匯出 Excel。

## 頁面分三步

- 第一步：框選與命名。畫面會顯示目前是否已有選區，並維護已保存調類標籤。
- 第二步：啟動分析。只有當已有保存調類時，第二步按鈕才有意義。
- 第三步：結果展示。會顯示最高值、最低值與 T 值結果圖，並開放 Excel 匯出。

## 資料保存行為

- 已保存的調類會寫進 localStorage，所以重新進頁後會自動恢復。
- 舊格式快取若還存在，元件掛載時會自動轉成新的 `segments` 結構。
- 清空全部會一次移除目前所有已保存調類。

## 匯出結果

- 匯出檔名會以「方音圖鑑_T值法定調_」開頭。
- 工作表名稱固定是「石峰T值分析」。
- 匯出內容是目前算出的 T 值分析結果，不是原始完整 Praat JSON。

## 常見坑位

- 若這次分析沒有 pitch 資料，頁面只會顯示無資料提示，三步流程都跑不起來。
- 沒有先框選資料，或沒填調名，就不能把樣本加入調類。
- 這頁的歷史調類會持久保存；若前一次測試資料還在，記得先清空再做新一輪比較。
