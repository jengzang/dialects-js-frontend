這一頁對應 /menu/source，用來查看公開資料來源表。

建議新手這樣用：
1. 先看表格上方標題與隱私連結。
2. 再用表格篩選省、市、縣、鎮或分區。
3. 最後看頁尾總筆數說明。

實際規則：
- 頁面載入時，會同步查兩個數字：可用地點數、資料總筆數，最後顯示在表格下方。
- 表格固定查詢 query 資料庫中的 dialects 表。
- 頁面內建 defaultFilter：只顯示「存儲標記 = 1」的資料，因此不是所有底層資料都直接公開列出。
- 可篩選欄位包含地圖集二分區、音典分區、省、市、縣、鎮；簡稱與來源欄位則不提供篩選。
- 上方的「隱私」連結會直接跳到 /menu/privacy。

容易踩坑：
- 你在這裡看到的是經過預設過濾後的公開資料，不是完整原始庫。
- 若只想查引用方式或使用規範，應直接點上方隱私連結。
