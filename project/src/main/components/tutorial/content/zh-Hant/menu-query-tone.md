# 查調類

這一頁是最簡單的查詢頁之一：不用先輸入字，也不用組中古條件，只要決定查哪些地點或分區，就能直接查看各地的調值 / 調類資料。

## 你在這一頁會看到什麼

- 這個分頁本身沒有額外的上方條件區。
- 主要可操作內容就是下方的地點 / 分區輸入元件。
- 按下運行後，系統會跳到結果頁，結果頁再以「查調」模式載入資料。

## 最適合新手的操作順序

1. 先選幾個你想看的地點，或先選一個分區。
2. 第一次使用時，建議不要一下子放太多地點，否則調值表會很分散。
3. 按運行後到結果頁查看調類、調值與不同地點的對應關係。

## 這一頁有哪些實際規則

- 這頁不需要額外選條件，真正的核心限制就是地點 / 分區。
- 執行時只會把 locations、regions、region_mode 帶到結果頁。
- 進結果頁後，會由結果頁呼叫查調 API，所以你在本頁按完按鈕後看到的是「跳轉」，不是本頁內部刷新。

## 新手怎麼用最不容易亂

- 若你只想知道某幾個點的聲調系統，直接輸入地點最快。
- 若你想看一整片區域的聲調分布，先選分區再讀結果。
- 如果資料太多，先把範圍縮小，再觀察調類名稱和調值，不要一開始就全域閱讀。

## 新手最常見的卡點

- 看不出差別：通常是地點太多，先縮小範圍。
- 不知道先看調類還是調值：可以先看分類名稱，再看具體數字。
- 以為這頁沒有內容：其實它是「用地點直接查調」的快捷入口，重點不在上半部，而在地點條件與結果頁。

## 什麼時候改用別的頁面

- 想比較兩個調類是否合流：改用「比較調類」。
- 想從字或中古條件切入：改用查字、查中古或查音位。
