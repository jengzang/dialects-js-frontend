# 比較漢字

把兩個漢字放在同一張地圖邏輯裡比較。這一頁最適合回答「甲字和乙字在各地方言裡，到底哪裡相同、哪裡不同」。

## 你在這一頁會看到什麼

- 左右各一個單字輸入框，分別代表組 1 和組 2。
- 中間下方要再選一個比較角度：聲母、韻母或聲調。
- 地點 / 分區仍然在頁面下半部共用。
- 按下比較後不去結果頁，而是直接跳到「地圖查看」頁。

## 最適合新手的操作順序

1. 先在組 1、組 2 各輸入 1 個字。
2. 再選比較角度：你是要比聲母、韻母，還是聲調。
3. 覺得範圍太大時，再補選地點或分區。
4. 按下比較後直接去地圖頁讀圖。

## 這一頁有哪些實際規則

- 兩個輸入框都只保留 1 個有效字；就算你貼了多個字，系統也只會留下前 1 個。
- 三個比較角度是單選，必須選一個。
- 只有「兩個字都已填」且「比較角度已選」時，比較按鈕才會啟用。
- 執行後會呼叫 compareChars，並把結果加工成地圖圖例，不經過結果頁中轉。

## 地圖頁上會看到什麼

- 地圖會把各地結果分成幾類：相同、部分相同、不同、未知。
- 所以這一頁的重點不是表格，而是空間分布。
- 如果你想先看某個字本身的詳細列表，再來比較，可先回查字頁確認。

## 新手最常見的卡點

- 按鈕是灰的：多半是第二個字還沒填，或比較角度還沒選。
- 結果太花、看不懂：先縮小到幾個地點，別一上來就全域比較。
- 兩個字差別很小卻看起來到處不同：先確認你選的是不是「聲調」，而不是「韻母 / 聲母」。

## 什麼時候改用別的頁面

- 想比兩個中古條件，不是兩個字：改用「比較中古」。
- 想看兩個調類是否合流：改用「比較調類」。
- 想先單獨查某個字：改用「查字」。
