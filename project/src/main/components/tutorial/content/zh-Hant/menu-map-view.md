這一頁對應 /menu/map/view，是地圖頁的主顯示區。

建議新手這樣用：
1. 先在其他查詢頁或「自定義」頁準備資料。
2. 切回「地圖」分頁查看結果。
3. 需要看原始地名時，切換「查看地名」；需要看自訂資料時，再開「自定義資料」。
4. 點地圖上的地點，可開啟該地點詳情彈窗。

實際規則：
- 地圖、分區、自定義三個分頁共用同一個頁面容器，切頁時會盡量保留狀態。
- 如果網址帶有 feature、locations、regions、regionMode 參數，頁面會自動載入對應自定義特徵，載入後再把這些參數從網址清掉，避免重新整理時重跑一次。
- 自定義特徵成功載入後，頁面會自動切到特徵模式，並自動開啟「自定義資料」。
- 在特徵模式下，頁面上方會出現特徵下拉選單；若只有一個特徵，會直接顯示成按鈕。
- 在比較模式下，上方顯示的是目前比較的對象名稱，不是特徵選單。
- 點地圖會把經緯度傳給下方的自定義資料面板；提交成功後，系統會自動開啟自定義資料並刷新目前圖層。

容易踩坑：
- 未登入時，不能打開「自定義資料」開關。
- 只有切到真正的地圖分頁時，網址中的 feature 查詢才會自動執行。
- 若目前沒有任何特徵資料，上方不會出現可選特徵。
