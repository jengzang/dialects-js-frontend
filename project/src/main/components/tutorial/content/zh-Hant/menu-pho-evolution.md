這一頁對應 /menu/pho/evolution，用來看單一地點在不同字表分類下的音變分布。

建議新手這樣用：
1. 先登入。
2. 選統計模式、字表、第一層分類、第二層分類。
3. 輸入 1 個地點。
4. 按查詢後，再切換「聲母 / 韻母 / 聲調」分頁觀看圖表。
5. 需要總覽流向時，可打開 Sankey 顯示。

實際規則：
- 這一頁要求登入；未登入按查詢時會先警告，再直接跳到 /auth。
- 地點上限是 1 個，程式也會再次檢查；超過 1 個會報錯，不會送出。
- 第一層與第二層分類都必須選，而且不能選成同一欄。
- 查詢成功後，圖表上方會出現「聲母、韻母、聲調」三個切換按鈕，每個按鈕後面帶該類別的圖數。
- 圓餅圖與 Sankey 是同一份資料的兩種視圖；切換 Sankey 只會重繪當前資料，不會重新查詢。
- 進入頁面時會先載入示例資料；如果你還沒查過真實資料，切換統計模式時也會跟著換示例資料。
- 在桌面端，滑過或點擊圓餅圖會顯示細節卡片；在手機端則以可關閉的卡片方式顯示。

容易踩坑：
- 看到圖不代表已經查到真實資料，因為此頁一開始就有示例圖。
- 真實查詢後，示例資料才會被正式結果取代。
