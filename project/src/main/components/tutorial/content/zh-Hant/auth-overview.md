# 賬戶總覽

這一頁是登入後的個人入口，不只是看帳號資訊，還負責在「概覽」與「排行榜」之間切換，並提供前往我的數據、我的分區、修改資料、登出的快捷按鈕。

## 你在這一頁會看到什麼

- 頂部會先顯示歡迎訊息。
- 上方有兩個主要 tab：概覽、排行榜。
- tab 切換器旁邊有一個 🎁 權益按鈕，可打開使用者權益說明。
- 概覽 tab 內會顯示帳號編號、註冊時間、在線時長，以及查詢統計卡片。
- 下方是一組操作按鈕：我的數據、我的分區、修改資料、登出；管理員還會多看到管理入口。

## 最適合新手的操作順序

1. 先停在概覽 tab，確認自己目前登入的帳號是不是正確。
2. 看一下帳號基本資訊與總查詢次數，確認資料是否正常同步。
3. 若想看細項，再展開查詢統計卡片。
4. 接著再按需求進入我的數據、我的分區，或修改資料。
5. 如果你只是想看站內活躍情況，再切到排行榜 tab。

## 這一頁有哪些實際規則

- auth 頁內部不是靠多個獨立頁切換，而是由 query `view` 控制：例如 overview、leaderboard、modify。
- 「排行榜」和「概覽」共用同一個 ProfileOverview 頁殼，只是切換內部 tab。
- 查詢統計卡片預設可折疊；展開後才會看到分類與細項。
- 管理入口只在 `user.role === 'admin'` 時出現，普通使用者看不到屬正常情況。

## 概覽 tab 該怎麼看

- 先看帳號編號、註冊時間、在線時長，這些是最基本的狀態資訊。
- 查詢統計先看總數即可；只有你真的要回顧自己常用哪些功能時，再展開細項。
- 下方功能按鈕才是這頁最常用的部分；很多時候你只是經過這裡去下一個頁面。

## 排行榜 tab 是做什麼的

- 這不是修改帳號的地方，而是切去看排行榜內容。
- 因為和概覽共用同一個頁殼，所以如果你覺得畫面很像，是正常設計，不是沒有切成功。

## 新手最常見的卡點

- 以為登入後沒有進總覽：其實有時只是還在初始化同步資料，等 loading 結束即可。
- 找不到統計細項：先點統計卡片右側的展開按鈕。
- 找不到管理入口：大多只是因為目前不是管理員帳號，不代表頁面故障。
- 切到排行榜後覺得沒換頁：這裡本來就是同頁殼內切 tab，不是整頁跳轉。

## 什麼時候去別的頁面

- 想看自己提交或收藏的資料：去「我的數據」。
- 想整理自己的分區設定：去「我的分區」。
- 想改名稱或密碼：去「修改資料」。
