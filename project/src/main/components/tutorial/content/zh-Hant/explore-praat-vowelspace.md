# Praat·元音空間

元音空間頁是根據分析結果裡的 F1 / F2 時序資料繪圖，不是獨立上傳頁。沒有共振峰資料時，這一頁就不會有可用圖形內容。

## 建議操作順序

1. 在設定裡至少勾選 formant 模組，再完成一次分析。
2. 進到元音空間頁後，先看目前是否有可用段落清單。
3. 先用預設的分段顯示瀏覽，再決定要不要切成全部散點顯示。
4. 需要和標準元音位置對照時，再勾選參考元音。

## 兩種顯示模式

- 分段顯示：只取 `rime_core`、`syllable_like`、`voiced` 這三類片段，並替每段配置顏色。
- 全部散點：直接把所有有效 F2/F1 點畫到圖上，不再按片段分色管理。
- 若這次結果裡根本沒有可用元音片段，頁面會強制退回全部散點模式。

## 分段模式可做什麼

- 預設全選所有可用段落，也可用「全選 / 全不選」快速切換。
- 每個段落會顯示時間範圍與類型標記。
- 下方統計卡片會按目前勾選的段落，顯示平均 F1、平均 F2 與資料點數。
- 圖上的顏色、段落卡片與統計卡片是一一對應的。

## 常見坑位

- 沒有 formant 資料時，這頁無法替你補算；要回上傳頁重跑分析。
- 參考元音只是疊加對照，不會改變原始分析點。
- 若把全部段落取消勾選，統計區只會提示目前沒有選中段落。
