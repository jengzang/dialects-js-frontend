這一頁對應 /menu/about/intro，是「關於」分頁中的預設首頁。

建議新手這樣用：
1. 先在這裡快速了解網站能做什麼。
2. 再按上方分頁切去「建議」、「喜歡」、「設定」。

實際規則：
- 「關於」頁本身是分頁頁面，不同分頁會切到 /menu/about/intro、/menu/about/suggestion、/menu/about/like、/menu/about/settings。
- 如果網址沒有帶分段，系統會把這一頁當成預設分頁。
- 這個分頁主要是靜態內容：功能簡介、作者感想、致謝清單，不會送出查詢。
- 頁面中的外部連結會直接開新分頁。

容易踩坑：
- 這頁是介紹入口，不是操作面板；真正的設定與回饋入口在同一組上方分頁裡。
