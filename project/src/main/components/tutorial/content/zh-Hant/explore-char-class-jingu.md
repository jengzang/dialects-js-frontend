# 字表·近古

近古分頁比中古、上古多一層：它不是只有單一資料表，而是同一分頁底下可切不同子表。網址仍是 `/explore/char-class?tab=jingu`，但還會再帶 `table` 與 `levels`。

## 建議操作順序

1. 先確認現在看的子表是哪一張，不要把不同子表的欄位混在一起理解。
2. 先選子表對應的預設分組，再做細部調整。
3. 如要比較不同子表，先記住目前網址參數，再切換子表看另一組配置。

## 這一頁有哪些子表

- `zhongyuan`：葉資料重點包含擬音，適合從韻母 / 小韻 / 聲調等角度看。
- `hongwu`：葉資料會帶上字、下字，註釋 joiner 是 `-`。
- `menggu`：葉資料可帶八思巴字、擬音、對應切韻音系音韻地位，註釋會用 ` / ` 串起來。
- 不同子表的可選層級與 preset 都不同，切表時頁面會重新正規化層級，不會硬沿用上一表的欄位。

## 可做的調整

- 左側可切子表、切 preset、增減層級、改層級順序。
- 右側可搜尋目前樹、切換註釋顯示、在失敗時重試同一組狀態。
- URL 會同步目前 `table` 與 `levels`，所以複製網址能直接分享同一視圖。

## 常見坑位

- 近古分頁切子表時，不只是資料內容變，整個可選欄位集合也會變。
- 某些葉資料顯示的是多欄位註釋拼接結果，不是單一欄文字。
