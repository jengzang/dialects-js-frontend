# 合併工具

這個工具是三步驟流程：先放參照表，再上傳待合併檔，最後下載合併結果。頁面本身有步驟指示器，完成前一步後才適合往下走。

## 步驟 1：準備參照表

- 參照表只接受 `.xlsx`、`.xls`，單檔上限 3 MB。
- 沒登入時，參照表不會上傳。
- 上傳成功後，畫面會顯示字數與欄位數。
- 也可以先看預設參照表；預覽後可直接下載，或一鍵套用成目前的參照表。

## 步驟 2：加入待合併檔

- 這一步可一次選多個檔案。
- 同樣只接受 Excel，單檔上限 3 MB。
- 最多 20 個檔案。
- 若有不合格檔案，頁面會提示，但仍會保留合法檔案。
- 每個檔案都可單獨移除。

## 步驟 3：開始合併

1. 按「開始合併」後，頁面會自動切到結果步驟。
2. 進度區會顯示百分比、已處理筆數與成功數。
3. 完成後可直接下載合併結果，也可按重設重來。

## 實際輸出

- 完成畫面會顯示本次合併了多少個檔案。
- 下載按鈕會輸出工具生成的合併結果檔。

## 常見坑位

- 沒有先準備參照表，就不能進到下一步。
- 若參照表被移除，前面的統計也會一起清掉。
- 檔案超過數量上限時，新的檔案不會再加入。
- 若原始資料品質不穩，建議先去數據檢查頁整理後再合併。
