這一頁對應 /menu/privacy，集中放引用建議、隱私政策與免責聲明。

建議新手這樣用：
1. 需要引用本站資料時，先看最上方引用格式。
2. 想直接帶走引用文字，就按複製按鈕。
3. 再往下讀隱私、Cookie、聯絡方式與免責聲明。

實際規則：
- 上方引用區有一個複製按鈕；按下後會把引用文字寫入系統剪貼簿，並跳出提示視窗。
- 隱私頁內容是純閱讀頁，沒有站內提交表單。
- 聯絡方式區的網站、GitHub、知乎連結都會在新分頁開啟。
- 這一頁同時也是 source 頁上方「隱私」連結的目的地。

容易踩坑：
- 複製成功提示用的是瀏覽器 alert，不是頁內小通知。
- 這一頁主要是規範與說明，不會顯示你的個人帳號資料。
