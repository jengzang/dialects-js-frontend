# 廣東自然村樹狀圖

這一頁用樹狀方式查看「广东省自然村」資料。頁面先只載入市級清單；每一個市的完整樹狀資料要按該市卡片上的「載入」後才會抓取，所以第一次打開時不會直接看到所有自然村。

## 這一頁有什麼

- 頁首標題、說明文字。
- 一個全頁搜尋框。
- 下方是一組市級卡片，每張卡片代表一個市。
- 每張市卡片都可能出現三種狀態：載入中、載入失敗、已載入。
- 載入完成後，卡片內會顯示樹狀節點。
- 點節點右側的 🌍 可以打開地圖彈窗。

## 資料結構怎麼看

這個樹會依序整理成多層級：
- 市級
- 區縣級
- 鄉鎮級
- 行政村
- 自然村

最底層自然村節點會把資料直接拼在名稱後面顯示：
- 村名
- 方言分布
- 若有經緯度，還會顯示 `(longitude,latitude)`

例如某個自然村若有座標，節點文字會是「村名 + 方言分布 + 座標」的形式。

## 建議操作順序

1. 先等待頁面把市級清單載入完成。
2. 找到要看的市，按該卡片上的「載入」。
3. 在樹中點擊節點本身或右側 ＋ 展開下一層。
4. 想快速定位時，按節點右側的 🌍 打開地圖。
5. 如果要跨很多市找詞，再使用頁首搜尋框。

## 搜尋怎麼運作

搜尋框會比對目前已經載入到前端的節點名稱。

重點是：
- 它不會自動替你載入尚未按過「載入」的市。
- 只要節點本身符合搜尋字串，就會保留該節點。
- 如果符合的是較深層子節點，上層路徑會自動展開，方便直接看到命中的位置。
- 如果是父節點本身符合，但子節點沒有被關鍵字命中，父節點會保留，但不一定自動展開。
- 搜尋結果為空時，卡片內會顯示沒有結果。

## 樹狀節點的互動

- 📁 代表還有下級資料。
- 📍 代表最底層節點。
- 點整個節點列，可以切換展開或收合。
- 有下級時，右側會有 ＋ 按鈕；展開後會旋轉成收合狀態。
- 搜尋命中的文字會被高亮。
- 若節點名稱剛好是 `(空)`，畫面上會改顯示成提示使用者展開查看的文字，而不是直接顯示 `(空)`。

## 地圖按鈕怎麼運作

每個節點右側都有 🌍：
- 如果你點的是分支節點，系統會遞迴收集它下面所有有資料的自然村，然後一起畫到地圖上。
- 如果你點的是最底層自然村，只會顯示這一筆村落。

地圖彈窗裡可以做這些事：
- 看目前有效座標的村落數量。
- 切換底圖樣式。
- 重設視角到目前村落範圍。
- 切換全螢幕。
- 如果資料裡有方言分布，還會出現顯示模式切換：
  - 村名模式：點位會做聚合。
  - 方言模式：只顯示有方言資料的村落，並以不同顏色標示不同方言，不做聚合。

## 你會看到哪些輸出

- 初始載入時，只看到市級名稱清單。
- 某市載入成功後，會出現已載入狀態與完整樹。
- 某市載入失敗時，卡片內會顯示錯誤訊息，並提供重試按鈕。
- 搜尋沒有命中時，已載入的該市卡片會顯示空結果提示。
- 地圖只會使用合法經緯度的村落資料；沒有有效座標的項目不會出現在地圖上。

## 常見容易誤會的地方

- 看不到村落，不一定是沒有資料；可能只是那個市還沒按「載入」。
- 搜尋不到資料，不一定是關鍵字錯；也可能是目標市尚未載入。
- 地圖上的點數可能少於樹上的葉節點數，因為地圖會過濾掉沒有合法經緯度的資料。
- 切到方言模式後，沒有方言分布的村落會被隱藏。
- 按 Esc 時，如果地圖已在全螢幕，會先退出全螢幕；再按一次才是關閉彈窗。

## 適合什麼情況使用

- 想沿著行政層級慢慢往下找某一批自然村。
- 想從某個區縣、鄉鎮或行政村，一次把下屬自然村畫上地圖。
- 想先用結構化方式縮小範圍，再做地圖比對。
