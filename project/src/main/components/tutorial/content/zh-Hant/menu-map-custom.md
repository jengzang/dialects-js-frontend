這一頁對應 /menu/map/custom，分成「查自己的自定義特徵」與「新增自己的自定義資料」兩條路。

建議新手這樣用：
1. 如果你想把自己做過的特徵畫上地圖：先登入，再選地點/分區，搜尋特徵名稱，選中後按查詢。
2. 如果你想新增單筆資料：按「逐條新增」，系統會跳到地圖頁並自動打開填寫面板。
3. 如果你想管理多筆資料：按「批量新增」，系統會進入個人資料管理頁。

實際規則：
- 特徵搜尋只有登入後才會真正向後端查詢；未登入時，搜尋框不會提供建議。
- 特徵輸入有 300ms 防抖；就算輸入空字串，登入狀態下也可能向後端取回可選特徵。
- 按「查詢」時，頁面不直接畫圖，而是把 feature、locations、regions、regionMode 寫進網址，然後跳到 /menu/map/view。
- 地圖頁收到這些網址參數後，才會真正載入對應特徵，並在完成後清掉網址參數。
- 按「逐條新增」時，系統會跳到 /menu/map/view?openPanel=true；地圖頁的自定義資料面板會自動展開。
- 按「批量新增」時，必須先登入；成功後會跳到 /auth/data，並帶上目前使用者名稱。
- 進入此分頁時，如果已登入，頁面會另外抓取你的自定義資料總數。

容易踩坑：
- 沒有先從建議清單選中特徵，只是手動打字，按查詢時仍可能被視為「未選特徵」。
- 查詢送出後，這一頁的特徵輸入會被清空，屬於正常行為。
- 幫助按鈕不是裝飾，裡面有欄位說明與蒐集格式示例，第一次新增資料前建議先看。
