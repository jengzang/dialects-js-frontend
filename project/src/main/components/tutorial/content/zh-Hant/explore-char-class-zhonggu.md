# 字表·中古

中古字表不是單純靜態表格，而是「頁面預設 + 層級配置 + 樹狀載入」的互動頁。這個分頁對應 `/explore/char-class?tab=zhonggu`，並會把目前表別與層級順序同步到網址參數。

## 建議操作順序

1. 先停在中古分頁，使用預設預設組合進頁。
2. 在左側先選預設分組，例如韻系、聲母、清濁等導向。
3. 再視需要增減層級，或調整層級順序。
4. 樹載入後，可在右側搜尋節點，並切換是否顯示註釋。

## 這一頁的預設內容

- 中古頁目前只有一張 `characters` 表。
- 可用層級包含攝、呼、等、韻、入、調、清濁、系、組、母、部位、方式。
- 預設組合不是任意字串，而是頁面內建 preset；切換後會直接改寫整組層級。
- 頁面會把 `table` 與 `levels` 寫回網址，所以重新整理後可回到相同配置。

## 可做的調整

- 新增層級：系統只會加入目前還沒使用過的下一個欄位，不會重複加同一層。
- 調整層級：每層都可改成別的欄位，也可上下移動順序。
- 刪除層級：至少要保留一層，所以剩最後一層時不能再刪。
- 搜尋：只是在目前已載入的樹上做前端查找，不是重新請後端做全文檢索。

## 常見坑位

- 切換到別的字表分頁再回來時，這頁自己的暫存樹與搜尋字串會被重置。
- 改網址中的 `levels` 後，頁面會先做正規化；無效欄位不會原樣保留。
- 若當前狀態載入失敗，可直接用頁面內的重試邏輯重新抓同一組狀態。
