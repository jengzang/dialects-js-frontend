這一頁對應 /menu/map/divide，用來把地點或分區直接畫成點位地圖。

建議新手這樣用：
1. 先輸入地點名稱，或從分區選擇器勾選分區。
2. 選好要顯示的分區層級。
3. 按下執行後，系統會直接跳回地圖頁顯示點位。

實際規則：
- 這一頁會把文字輸入的地點、以及自訂分區展開後的地點合併，再一起送出查詢。
- 送出的參數包含 locations、regions、region_mode；region_mode 會依你目前選的是地圖集分區還是音典分區而改變。
- 查詢成功後，系統會把地圖模式切成 dot，並自動跳到 /menu/map/view。
- 這個下拉選單同時會把分區層級回傳給主頁，所以回到地圖後，點位顯示會跟著目前層級走。
- 已登入用戶查詢時會把個人自定義資料一併納入；匿名用戶不會。

容易踩坑：
- 匿名狀態按執行時，系統只會警告「沒有自定義資料」，不會阻止查詢；公開資料仍然會照常出圖。
- 這一頁的結果不是留在本頁，而是直接跳回地圖頁顯示。
- 如果只選了分區但沒注意分區模式，畫出來的範圍可能和你預期不同。
