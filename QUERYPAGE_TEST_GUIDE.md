## 常见问题排查

### 问题 1：文本没有翻译
**可能原因**：翻译 key 不存在
**解决方案**：检查 `src/i18n/locales/*/query.json` 文件

### 问题 2：语言切换后文本不更新
**可能原因**：组件没有使用响应式翻译
**解决方案**：确认使用了 `$t()` 或 `t()`

### 问题 3：动态文本显示错误
**可能原因**：插值变量名不匹配
**解决方案**：检查 `$t('key', { var: value })` 的变量名
